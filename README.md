# CNY Game 2020 backend
Author: Lee Ming Liang <br>
This project is build using SpringBoot using embedded tomcat. <br>
#### Technologies 
- JWT authentication
- Websocket
- Rest Api 
- Hibernate/JPA 
- SQL 


## Setting up 
###  Prerequisites   
- Java 11
- Intellj IDE 
- Maven 
- MYSQL 8 

### Parameters 
- Change the mysql connection url, username and password in application.properties
